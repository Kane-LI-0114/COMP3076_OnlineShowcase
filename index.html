Group 1 Cloud Fauna Transformation 2025

This project is part of my ongoing work on generative vision systems that merge classical computer vision with diffusion models. The goal is to reshape cloud regions in photographs into clear animal silhouettes while keeping their native cloud texture and lighting. The method—Cloud-Oriented Style Transfer (COST)—follows four steps:
  1. Detect and segment the sky area with a fine-tuned object detector trained on a custom cloud dataset;
  2. Enhance the edges and detail in the selected region to create a reliable mask;
  3. Guide a Stable Diffusion model with the mask and a chosen animal outline so the generated form matches the original cloud’s translucency and scale;
  4. Post-process for color, texture, and boundary consistency to ensure the new shape blends with foreground objects and weather conditions.
COST is tested across clear skies, complex foregrounds, partial occlusions, and varied cloud densities, demonstrating context-aware integration without disrupting scene semantics.
